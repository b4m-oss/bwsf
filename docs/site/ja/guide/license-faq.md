# ライセンスに関するよくある質問

<AccordionList>
  <AccordionQA title="`bwsf`は、無料で使えるのですか？">

はい、`bwsf`は、`MIT License`によって公開されており、誰でも無料で利用することができます。

  </AccordionQA>

  <AccordionQA title="`bwsf`を使用して、損害が発生した場合は、誰が責任をとるのですか？">

`bwsf`は、`MIT License`に基づいて公開されているため、開発者は責任を一切取ることができません。

詳しくは、[MIT Licenseの条文](./license.md)をご覧ください。

従って、損害発生時の責任は開発者以外のステークホルダーで明示的にするようお勧めいたします。

  </AccordionQA>

  <AccordionQA title="`bwsf`を、自社の製品に組み込むことは可能ですか？">

はい、可能です。[MIT License](./license.md)の範囲化で組み込むことができます。

また、損害発生時の責任分解については、一つ上の質問と同じです。

  </AccordionQA>

  <AccordionQA title="`bwsf`を、自社の製品に組み込んだ場合、そのソースコードは公開する必要がありますか？">

必要はありません。[MIT License](./license.md)では、そのような行動を義務化していません。

他のオープンソースライセンスには、そういったものもありますが、MIT Licenseでは義務化されていません。

  </AccordionQA>

  <AccordionQA title="`bwsf`に機能を追加したいのですが">

問題ありません。GitHubからフォークして、開発していただければ結構です。

追加した機能をプルリクエストの形で送っていただければ、追加された機能を`bwsf`の新機能として取り込むことも可能です。

  </AccordionQA>

  <AccordionQA title="`bwsf`に機能を追加して欲しいです。自分ではできないのですが。">

`bwsf`はオープンソースプロジェクトです。

まずは、[私たちのGitHubディスカッション](https://github.com/b4m-oss/bwsf/discussions/categories/ideas)、新機能追加要望を投稿して下さい。

新機能を追加するかどうかは、開発チームで検討します。

新機能の追加開発が決定した場合、その新機能実装がいつになるのかは約束されません。

  </AccordionQA>

  <AccordionQA title="開発費用を支払うので、新機能を追加してもらえますか？">

そういった機能追加の形にも対応しておりますが、ご相談内容とこちらの条件が折り合うか次第です。

まずは、[知的・自転車のお問い合わせフォーム](https://b4m.co.jp/contact)からご連絡ください。

  </AccordionQA>

  <AccordionQA title="`bwsf`に対して、寄付をしたいのですが">

お気持ちありがとうございます。

[当社のお問い合わせフォーム](https://b4m.co.jp/contact)からご相談ください。

  </AccordionQA>
</AccordionList>
