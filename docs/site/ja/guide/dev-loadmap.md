# 開発予定中の機能

- `dotenvs`フォルダ以外の名称利用可能化
- `bwsf.config.json`ファイルの機能実装
  - `ignore`や`override_project_name`などの設定が可能に
- `bwsf clean`コマンド： 現在プロジェクトからの、.envファイルの削除。
- .gitフォルダによる、プロジェクトルートの確認（間違ったディレクトリでのコマンド実行を阻止）

# バージョニングについて

`bwsf`は、セマンティック・バージョニングを採用しています。

現在、v1.0.0のリリースは未定です。

v0.x.0のアップデートは、常に破壊的変更が考えられます。

プロダクションでの利用は、十分ご注意ください。
